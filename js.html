História


  • JavaScript foi desenvolvida pela Netscape para trabalhar com aplicações interativas em páginas (X)HTML;
  • Depois do seu sucesso inicial, recebeu uma contribuição considerável da Sun Microsystems e, assim, é uma linguagem baseada em Java;
  • Suporta a maior parte das sintaxes e dos comandos da linguagem Java;
  • Uma das desvantagens é que suporta poucos tipos de dados e implementa apenas alguns conceitos de Orientação a Objetos;

Introdução


  • Javascript oferece rapidez no desenvolvimento e é a mais popular da Internet;
  • Projetada para adicionar interatividade nas páginas Web;
  • É uma linguagem de script, ou seja, pode ser invocada dentro de outras linguagens de programação;
  • Incorporada diretamente nas páginas HTML;
  • Importante: JavaScript não é Java!!!

Pág. estática Vs Pág. dinâmica


    Página estática


  • Construídas usando apenas HTML;

  • Além dos links, possuem poucos efeitos e funcionalidades e;
  • Fáceis de serem criadas.

Pág. estática Vs Pág. dinâmica


    Página dinâmica


  • Podem incluir qualquer tipo de efeito especial ou funcionalidade e;

  • É necessário utilizar outras linguagens de programação.

Aplicações em JavaScript


  • Criação de páginas com efeitos especiais e com interatividade;
  • Detecção do navegador do visitante;
  • Reação a eventos como resposta às ações do usuário;
  • Validação de dados;
  • Criação desde pequenos scripts, até programação orientados a objetos, com funções, estruturas de dados complexas e outras.

Algumas características


  • É case sensitive;
  • É fracamente tipada (ou dinamicamente tipada);
  • O navegador do cliente é o responsável por interpretar as instruções;
  • O ponto e vírgula: Opcional no final da marcação, e permite várias instruções na mesma linha.

    Declaração de variáveis


    • Utiliza-se o termo var (opcional) e os nomes devem começar com uma letra ou com um caractere sublinhado: 
    
var idade;
var saudacao;
    • Pode receber um valor já na declaração: 
    
var idade=5;
var saudacao ="Olá!";

    Operador "+"


    • Quando usado entre strings, devolve uma string: 
    
< script type="text/javascript">
      var res = "Oi, " + "José"; //Armazena a string "Oi, José"
< / script >
    • Quando usado entre strings e números, devolve uma string: 
    
< script type="text/javascript">
      var res = "José" + 10; //Armazena a string "José10"
< / script >

    Operador "+"


    • Quando usado entre números, devolve a soma entre eles: 
    
< script type="text/javascript" >
      var res = 10 + 10; //Armazena o valor numérico 20
< / script >

    Operadores lógicos


    Operador condicional


  • Sintaxe: 
    • 
nomeVariavel = (condição) ? valor1 : valor2;

  • Exemplo: 
    • 
resultado = (idade >= 18) ? "MAIORIDADE" : "MENORIDADE";

    Operador condicional composta

  • Sintaxe: 
    • 
if (condição) {
    //código que será executado se a condição for verdadeira
}
else {
//código que será executado se a condição for falsa
}
  • Exemplo: 
    • 
var idade = 20;
if (idade >= 18) {
    document.write("MAIORIDADE");
}
else {
document.write("MENORIDADE");
}

    Operador condicional alinhada

  • Sintaxe: 
    • 
if (condicao_01) {
    //código que será executado se condicao_01 for verdadeira
}

else 
    if (condicao_02){
    // código que será executado se condicao_02 for verdadeira
}

else {
  // código que será executado se condicao_01 e condicao_02 forem falsas
}

    Switch

  • Sintaxe: 
    • 
switch (nomeVariavelAnalisada) {
case 1:
    //executa codigo do bloco 1
    break;

    case 2 :
    //executa codigo do bloco 2
    break;

    default:
    codigo para ser executado se nomeVariavel é diferente do caso 1 e 2
}

    Estrutura de repetição: For


  • Sintaxe: 
    • 
for (nomeVar=valorInicial; nomeVar <= valorFinal; nomeVar = nomeVar + inc) {
    //código a ser executado
}

  • Exemplo: 
    • 
for (i=1; i <= 10; i++) {
    document.write("i = " + i + "
    ");
}

    Estrutura de repetição: While


  • Sintaxe: 
    • 
while (condicaoParada) {
    //código a ser executado
}

  • Exemplo: 
    • 
i = 1;
while (i <= 10) {
    document.write("i = " + i + "
    ");
    i++;
}

    //código a ser executado

    Caracteres Especiais


    Funções


    • Uma função contém código que será executada por um evento ou por uma chamada a ela;
    • Pode chamar uma função de qualquer lugar da página (ou de outras páginas se a função estiver em um arquivo externo .js);
    • Podem ser definidas no < head > e na seção < body >;
    • Colocando no garante que ela seja carregada no browser antes de ser chamada;

    Exemplo


  • Na tag < head >, copie e cole o código abaixo: 
    • 
< script type="text/javascript">
    function mostraMensagem(){
        alert("Evento onclick do botão foi disparado!");
    }		
< / script >

  • Na tag < body >, copie e cole o código abaixo: 
    • 
< form >
    < input type="button" value="Evento onclick" onclick="mostraMensagem()" >
< / form >

    Declaração Return

  • Essa declaração é usada para especificar um valor a ser retornado pela função: 
    •

    Objeto Error


    • O objeto Error contém informações dos erros gerados durante a execução de um JavaScript;
    • A propriedade name retorna o nome do erro;
    • Já a propriedade message apresenta a descrição do erro gerado;
    • O método toString() concatena as informações geradas pelas propriedades name e message.

    Tratamento de erros

    • Para realizar o tratamento de erros em JavaScript, podemos utilizar o bloco de instrução try/catch/finally: 
    
try {
   //tentativa de execução de instruções suspeitas
}

catch (e) {
    document.write(e.toString());
    //document.write(e.name + " –> " + e.message);
}

finally {
  //instrução que será executada independente de ocorrer erros
}

    A palavra-chave throw

    • Permite que sejam criados erros customizados e, quando usada com try/catch, o fluxo do programa pode ser controlado.
